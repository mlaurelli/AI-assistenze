DESL - Definisci sollecitazioni

Ambiente: Progettazione interattiva sezioni

Modificatori: + -

Definisce le sollecitazioni.

Sintassi del comando:

DESL

Selezionare le aste

oppure:

DESL

oppure:

DESL F / Nome del file delle sollecitazioni

Nella prima forma si possono definire le sollecitazioni selezionando un'asta che abbia la stessa sezione. Questo funziona solamente se la progettazione interattiva è stata avviata con il menu a comparsa con il tasto destro sulle aste oppure con il comando EDIT SEZ A.

Nella seconda forma si possono definire le cinque componenti di sollecitazione.

Nella terza forma attraverso il modificatore F si può specificare il nome del file, in formato CSV, da cui leggere le sollecitazioni. Le sollecitazioni devono essere espresse in daN e m.

Note:

  • Nella definizione delle sollecitazione è possibile stabilire il tipo di normativa da utilizzare: tensioni ammissibili, stati limite.

  • Le sollecitazioni possono essere lette dal file in formato CSV solo nel caso di progettazione interattiva armatura sezioni con sollecitazioni inserite manualmente. Nel file devono essere contenuti i seguenti dati:

  1. Il tipo di normativa da utilizzare: TA (Tensioni Ammissibili), SLU (Stato Limite Ultimo), SLU S (Stato Limite Ultimo Sismico per calcoli eseguiti secondo il D.M. 16/01/96), SLO (Stato Limite di Operatività), SLD (Stato Limite di Danno), SLV (Stato Limite di salvaguardia della Vita), SND (Stato limite di salvaguardia della vita Non Dissipativo), SLC (Stato Limite di Collasso), SLE R (Stato Limite di Esercizio per combinazioni Rare), SLE F (Stato Limite di Esercizio per combinazioni Frequenti), SLE Q (Stato Limite di Esercizio per combinazioni Quasi permanenti), SLU I (Stato Limite di resistenza al fuoco);

  2. I valori delle cinque componenti di sollecitazione: N, Ty, Tz, My, Mz.

Da notare che i dati devono essere espressi in daN e m, separati da virgola o da punto e virgola, e non è possibile utilizzare normativa diverse. Un esempio di file dati potrebbe essere il seguente:

SLU, -10800, 0, -900, -1800, 0

SLE R, -7000, 0, -900, -1800, 0

SLE F, -7500, 490, 0, 0, 860

SLE Q, -1000, 0, -800, -1800, 0
