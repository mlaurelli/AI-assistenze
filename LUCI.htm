LUCI - Visualizza lucido di costruzione

Ambiente: Modellazione

Modificatori: ? + -

Attiva o disattiva la visualizzazione di un file LUC (file DXF/DWG/IFC convertito in file lucido) da usare come lucido di costruzione.

Sintassi del comando:

LUCI Nome del lucido o parametro / Spostamento X,Y,Z da assegnare

Nome del lucido o parametro

è il nome del file (con estensione LUC) precedentemente convertito dal file DXF/DWG/IFC o il parametro (+,-,X,Y,Z,?) di visualizzazione da utilizzare.

Spostamento X,Y,Z da assegnare

inserire il valore dello spostamento X,Y,Z da assegnare al lucido.

Note:

  • Per disattivare la visualizzazione del lucido occorre eseguire il comando LUCI -.

  • Il comando LUCI + attiva l'ultimo lucido utilizzato.

  • I comandi LUCI X , LUCI Y e LUCI Z spostano il lucido nella direzione indicata di una quantità specificabile.

  • Il comando LUCI ? apre una finestra di dialogo dalla quale è possibile selezionare il lucido da utilizzare.

Argomenti correlati: Visualizzazione di un lucido, Trasformazione file DXF/DWG in lucido, Trasformazione file IFC in lucido
