INSE - Inserisce un disegno Ms-Cad

Ambiente: Ms-Cad

Modificatori: L

Inserisce nel disegno le entità grafiche memorizzate in un file Ms-Cad o DXF/DWG.

Sintassi del comando:

INSE Nome del file Ms-Cad / Fattore di scala / Mantengo le altezze originali dei testi / Punto di inserimento / Angolo o secondo punto o P per primo punto / Angolo o primo punto

Nome del file Ms-Cad

è il nome del file Ms-Cad o DXF/DWG contenente il disegno da inserire.

Fattore di scala

è un fattore moltiplicativo delle entità grafiche contenute nel file Ms-Cad o DXF/DWG.

Mantengo le altezze originali dei testi

specificare se mantenere le altezze originali dei testi o scalare anche quelle con il fattore di scala.

Punto di inserimento

indica la posizione di inserimento del disegno.

Angolo o secondo punto o P per primo punto

indica l'angolo di orientamento del disegno rispetto all'orizzontale che può essere inserito mediante un valore numerico o individuato dal vettore congiungente il punto di inserimento e il secondo punto o permette di modificare il punto di origine del vettore.

Angolo o primo punto

indica il valore numerico dell'angolo di orientamento del disegno o il primo punto del vettore.

Note:

  • Se nel file sono contenute delle entità testo e se il fattore di scala è diverso da 1 o se il file era stato creato in una scala diversa da quella attuale ModeSt chiede se devono essere mantenute le altezze dei testi. In caso affermativo i testi vengono riscalati in modo che al momento del plottaggio mantengano l'altezza originale, in caso contrario l'altezza di plottaggio verrà moltiplicata per il fattore di scala specificato.

  • Con il modificatore L (Link) il disegno viene inserito in modo dinamico anzichè statico, come specificato in Inserimento disegni.

  • ModeSt effettua in modo automatico una trasformazione dei Layer presenti nel file DXF/DWG in modo da adattarla alla definizione di layer di ModeSt, nel caso di layer non corrispondenti alle entità grafiche vengono assegnati progressivamente i layer di ModeSt da 1 a 6 ricorsivamente.

Sinonimi: INSERISCI, INSERT
