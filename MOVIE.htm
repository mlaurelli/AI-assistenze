MOVIE - Movie

Ambiente: Modellazione

Modificatori:

Il comando permette di creare filmati in formato AVI tramite l'assemblaggio di fotogrammi o per effettuarne la loro riproduzione.

Sintassi del comando:

MOVIE Azione del registratore / Nome del file

Azione del registratore

rappresenta la modalità di visualizzazione della geometria della struttura:

  • REC avvia la registrazione di un filmato

  • ADDFRAME aggiunge un fotogramma al filmato in registrazione

  • PLAY riproduce un filmato

  • STOP termina una registrazione o una riproduzione di un filmato

Nome del file

nome del file del filmato da creare o da riprodurre.

Note:

  • Sotto la cartella di installazione di ModeSt è presente una cartella di nome TMP che è per default la cartella dove vengono memorizzati i file dei filmati creati. Comunque è possibile salvare il file in qualunque allocazione specificando il percorso completo (ad esempio per salvare il file del filmato con nome PROVA nella cartella LAVORI occorre specificare: C:\LAVORI\PROVA).
