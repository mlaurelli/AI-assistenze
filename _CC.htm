?CC - Informazioni combinazioni condizioni di carico elementari

Ambiente: Modellazione

Modificatori:

Visualizza la tabella delle combinazioni delle condizioni di carico elementari presenti nel calcolo per i tipi di sollecitazioni che le prevedono.

Sintassi del comando:

?CC

Note:

  • Nella tabella vengono evidenziate con colore celeste le celle contenenti valori diversi da zero. Questo permette di individuare facilmente quali sono le CCE realmente presenti in una CC.

  • Nella tabella è possibile modificare solo combinazioni lineari ed inserirne di nuove solo se nel calcolo è stata effettuata almeno un'analisi lineare. Non è possibile inserire combinazioni di tipo non lineare.

  • Nel caso in cui la tabella delle combinazioni sia particolarmente complessa, può essere salvata in un file in formato ASCII per poter essere modificata e successivamente riletta dal file. Il file avrà lo stesso nome della struttura in esame ed estensione CMB. Il file può essere creato e scritto anche manualmente nel formato indicato nel capitolo Combinazione delle condizioni di carico.

Comandi correlati: ?CCE elenca le CCE.

Argomenti correlati: Sollecitazione e risultato corrente, Combinazione delle condizioni di carico
