CAMB - Cambia le entità grafiche

Ambiente: Ms-Cad

Modificatori:

Il comando permette di modificare i dati relativi ad un'entità grafica.

Sintassi del comando:

CAMB

Selezionare le entità grafiche

Layer / Altezza / Testo

I dati modificabili sono:

per l'entità LINEA

Layer

per l'entità CERCHIO

Layer

per l'entità TESTO

Layer, Altezza, Testo

Risulta comunque più agevole modificare i dati di una entità grafica utilizzano il menu a comparsa sull'entità con l'opzione Proprietà.

Sinonimi: CAMBIA, CHANGE
